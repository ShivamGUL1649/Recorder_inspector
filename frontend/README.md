# AI Playwright Smart Recorder

Lightweight zero-backend Playwright recorder and smart locator inspector.

This version runs fully with:

- GitHub Pages
- React/Vite frontend
- Bookmarklet-based recorder script
- No Node.js backend
- No Playwright server runtime
- No database
- No cloud VM

## Features

- Smart Inspector popup
- Draggable recorder panel
- XPath locator generation
- CSS selector generation
- Playwright locator generation
- Element Pick stays ON until manually disabled
- Record click actions
- Record fill actions
- Record select dropdown actions
- Record checkbox/radio actions
- Generate Playwright script
- Copy locator/script directly
